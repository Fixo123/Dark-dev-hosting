## 📖 About

**DARK DEV HOSTING** is a free, cloud-native deployment platform that allows you to deploy any **public GitHub repository** to **Cloudflare Pages** with just one click. No configuration needed. Just paste your repository URL and deploy!

> ⚠️ **Note:** Currently supports **public repositories only**. Private repository support coming soon with GitHub OAuth integration.

---

| Feature | Status | Description |
|---------|--------|-------------|

## ⚡ Tech Stack

### Frontend
- **HTML5** - Structure

- **JavaScript** - Interactivity

- **Google Fonts (Orbitron)** - Fonts

- **Node.js** - Runtime environment
- **Express.js** - Web framework
- **Render** - Hosting platform
- **Cloudflare Pages** - Deployment target

- **GitHub API** - Repository verification
- **Cloudflare API** - Pages deployment
